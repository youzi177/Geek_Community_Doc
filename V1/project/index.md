# 项目介绍

极客社区(Geek_Community)是一个全平台应用，主要覆盖PC、Android、IOS、鸿蒙、小程序等平台应用，不仅如此，还同时开源后台管理系统(Admin)和后端应用

## 为什么是社区类型的项目

在几乎所有的前端教学视频上，几乎都是以电商为主的前端或者中后台应用项目，但是这些应用也只能当“玩具”类型，不具备上线的可能性，电商最主要的就是支付功能，但是这点几乎就卡了很多人，因为对接微信支付和支付宝支付都是需要企业营业执照，还会有税务问题，要不就需要对接第三方，但是资金不安全，费率也贵，但是无论怎样，电商项目我个人看来是一种“烂大街”项目，而且真正具备上线的电商项目，逻辑业务后端接口数据库设计都是非常复杂的

另外：开源热度高的电商项目可以参考：[crmeb](https://www.crmeb.com/xiazai?from=vue) ，这是一个真正意义上的开源具备上线的项目，反观社区类型，有静态博客的hexo、还有Halo、WordPress，这些都是很优秀的项目，但是缺乏跨端应用，特别是Halo，前端确实很好，但是确一直找不到移动端的解决方案，手机浏览器虽然能看，但是体验远远差于PC，这也是为什么要做多平台应用

## 特点

全平台应用、持续维护更新迭代

## 如何阅读

开发过程按照功能模块开发，先开发后端接口，再开发前端模块，直到前后端项目完成，再完成后台管理系统Admin，后续的Flutter、鸿蒙、小程序这些其实都只是平台不同的写法，UI基本一致，功能一致，对接接口一致，可以并排开发。
