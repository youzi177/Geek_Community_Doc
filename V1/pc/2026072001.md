# 长列表的最后一页和请求锁

本章要解决3个问题：

1. 判断页面是否已经存在文章列表，不存在就直接把后端的list文章列表填充到数组，这通常是第一次进入网站的时候发生，存在的话，就在数组追加服务端返回的list文章列表，这通常是点击下一页的时候发生，如果不处理这一步，直接赋值，就会发生点击下页是在替换掉原来的数据
2. 判断后端的list文章列表是不是已经最后一页了，是的话，就要把更多求解换成没有更多了，让用户不要点击
3. 在网络非常卡顿的时候，如果多次点击更多求解，页面会出现bug，这个时候就需要加入请求锁，判断用户这次请求是否完成，没有完成的，就不要打开锁，否则在打开

## 第一个问题:

```ts
//数据
const query = reactive({
  status: '', //状态
  tag: '', //标签，精华
  sort: 'created',
  page: 0, //第几页
  limit: 10, //一页显示10条
  catlog: '', //分类
  lists: [
  ], //文章详情 为了演示，这里删除文章数据，如需要可以在上一章复制
})
const { status, tag, sort, lists } = toRefs(query)
//获取文章列表
const _getList = async () => {
  const options = {
    catalog: query.catlog,
    isTop: 0,
    page: query.page,
    limit: query.limit,
    sort: query.sort,
    status: query.status,
  }
  try {
    const result = await getList(options)
    //明确告知result就是HttpResponse类型
    const { code, data } = result as HttpResponse
    if (code === 200) {
      //判断页面有没有数据，没有就直接赋值
      if (query.lists.length === 0) {
        query.lists = result.data
      } else {
        //页面有数据就concat追加
        query.lists = query.lists.concat(data)
      }
    }
  } catch (error) {
    console.log(error);
  }
}
```

- 没有数据就`query.lists = result.data`
- 有数据就 `query.lists = query.lists.concat(data)`

## 第二个问题

```ts
//数据
const query = reactive({
  status: '', //状态
  tag: '', //标签，精华
  sort: 'created',
  page: 0, //第几页
  limit: 10, //一页显示10条
  catlog: '', //分类
  lists: [
  ], //文章详情 为了演示，这里删除文章数据，如需要可以在上一章复制
  isEnd: false, //是否最后一页
})
//获取文章列表
const _getList = async () => {

  const options = {
    catalog: query.catlog,
    isTop: 0,
    page: query.page,
    limit: query.limit,
    sort: query.sort,
    status: query.status,
  }

  try {
    const result = await getList(options)
    //明确告知result就是HttpResponse类型
    const { code, data } = result as HttpResponse
    if (code === 200) {
      //如果小于一页的数据，就是最后一页了
      if (data.length < query.limit) {
        query.isEnd = true
      }
      //判断页面有没有数据，没有就直接赋值
      if (query.lists.length === 0) {
        query.lists = result.data
      } else {
        //页面有数据就concat追加
        query.lists = query.lists.concat(data)
      }
    }
  } catch (error) {
    console.log(error);
  }
}
```

- 加入变量isEnd，通过判断`data.length < query.limit`来设置是否最后一页
- 然后需要把isEnd传递给ListItem，然后ListItem接收到这个变量，根据这个变量去判断显示更多求解还是已经到底线了

## 第三个问题

就是给请求加上请求锁，防止在网络环境很差的时候疯狂点击下一页更多求解去发送大量请求

```ts
const query = reactive({
  status: '', //状态
  tag: '', //标签，精华
  sort: 'created',
  page: 0, //第几页
  limit: 10, //一页显示10条
  catlog: '', //分类
  lists: [
    {
      uid: {
        isVip: 8,

      },

      isEnd: '0',

      status: '0',
      isTop: '0',
      tags: [
        {
          name: '精华',
          class: 'layui-bg-red',
        },
        {
          name: '热门',
          class: 'layui-bg-blue',
        },
      ],
      sort: '0',
      answer: '100',
    },
    {
      uid: {
        isVip: 8,
        name: '小封鸭',
      },
      title: '帖子的标题',
      content: '',
      created: '2026-7-20 00:00:00',
      catalog: 'ask',
      fav: '40',
      isEnd: '0',
      reasds: '10',
      status: '0',
      isTop: '0',
      tags: [
        {
          name: '精华',
          class: 'layui-bg-red',
        },
        {
          name: '热门',
          class: 'layui-bg-blue',
        },
      ],
      sort: '0',
      answer: '100',
    },
  ], //文章详情
  isEnd: false, //是否最后一页
  isRepeat: false, //请求锁
})
const { status, tag, sort, lists } = toRefs(query)
//获取文章列表
const _getList = async () => {
  if (query.isEnd) return
  const options = {
    catalog: query.catlog,
    isTop: 0,
    page: query.page,
    limit: query.limit,
    sort: query.sort,
    status: query.status,
  }
  if (query.isRepeat) return //判断是不是锁上了，true锁上，直接return，否则的话继续执行
  try {
    query.isRepeat = true //请求之后把请求锁 锁上
    const result = await getList(options)
    //明确告知result就是HttpResponse类型
    const { code, data } = result as HttpResponse
    if (code === 200) {
      query.isRepeat = false //数据请求完成之后，把锁打开
      //如果小于一页的数据，就是最后一页了
      if (data.length < query.limit) {
        query.isEnd = true
      }
      //判断页面有没有数据，没有就直接赋值
      if (query.lists.length === 0) {
        query.lists = result.data
      } else {
        //页面有数据就concat追加
        query.lists = query.lists.concat(data)
      }
    }
  } catch (error) {
    //console.log(error);

    //错误了也要先把锁打开
    query.isRepeat = false
  }
}
```
