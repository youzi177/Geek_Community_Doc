# 对接验证码接口

本章对接后端的验证码接口，使用axios发送请求，使用v-html显示后端传回来的svg

## 安装使用axios

安装

```
npm i axios -s
```

使用：（示例代码非完整）

```vue
<template>
<span class="svg" style="color: #c00" @click="_getCode" v-html="svg"></span>
</template>
<script lang="ts" setup>
import axios from 'axios'
import { onMounted, reactive, toRefs } from 'vue'
const state = reactive({
  username: '',
  password: '',
  code: '',
  svg: '',
})
const { username, password, code, svg } = toRefs(state)
const login = async () => {}
onMounted(() => {
  axios.get('http://localhost:3000/public/getCaptcha').then((res) => {
    state.svg = res.data.msg
  })
})
const _getCode = () => {
  axios.get('http://localhost:3000/public/getCaptcha').then((res) => {
    state.svg = res.data.msg
  })
}
</script>
```

- 简单就是页面挂载的时候请求接口，点击验证码的时候请求接口

- 对也svg微调一下样式，在App.vue下

  ```css
  .svg svg {
    position: relative;
    top: -12px;
    height: 50px;
  }
  ```

## 封装验证码接口

每个组件都需要重复以下代码，显然不够优雅：

```ts
onMounted(() => {
  axios.get('http://localhost:3000/public/getCaptcha').then((res) => {
    state.svg = res.data.msg
  })
})
const _getCode = () => {
  axios.get('http://localhost:3000/public/getCaptcha').then((res) => {
    state.svg = res.data.msg
  })
}
```

src目录下创建api目录，存放前端的接口

```ts
// src\api\login.ts
import axios from 'axios'
const getCode = async () => {
  try {
    const response = await axios.get('/public/getCaptcha')

    if (response.status === 200) {
      return response.data
    }

    return null
  } catch (error) {
    console.error(error)
    return null
  }
}
export { getCode }
```

设置axios的base URL

```ts
// src\main.ts
axios.defaults.baseURL =
  import.meta.env.MODE === 'development' ? 'http://localhost:3000' : 'https://you.domain.com'
```

使用：

```ts
onMounted(() => {
    _getCode()
})
const _getCode = () => {
    getCode().then((res) => {
        state.svg = res.msg
    })
```
