# 对接验证码接口

本章对接后端的验证码接口，使用axios发送请求，使用v-html显示后端传回来的svg

## 安装使用axios

安装

```
npm i axios -s
```

使用：（示例代码非完整）

```vue
<template>
<span class="svg" style="color: #c00" @click="_getCode" v-html="svg"></span>
</template>
<script lang="ts" setup>
import axios from 'axios'
import { onMounted, reactive, toRefs } from 'vue'
const state = reactive({
  username: '',
  password: '',
  code: '',
  svg: '',
})
const { username, password, code, svg } = toRefs(state)
const login = async () => {}
onMounted(() => {
  axios.get('http://localhost:3000/public/getCaptcha').then((res) => {
    state.svg = res.data.msg
  })
})
const _getCode = () => {
  axios.get('http://localhost:3000/public/getCaptcha').then((res) => {
    state.svg = res.data.msg
  })
}
</script>
```

- 简单就是页面挂载的时候请求接口，点击验证码的时候请求接口

- 对也svg微调一下样式，在App.vue下

  ```
  .svg svg {
    position: relative;
    top: -12px;
    height: 50px;
  }
  ```

- 其他组件也一样。这个请求可以封装为hooks，但是这里先这样，后续再封装
