# 对接邮箱服务接口

学习本章之前，最好已经开发好后端的邮箱接口才能进行对接

## 创建接口

打开：`src\api\login.ts`

```ts
//新增接口
import type { LoginInfo } 
const forget = async (option: LoginInfo) => {
  try {
    const response = await axios.post('/login/forget', { ...option })
    if (response.status === 200) {
      return response.data
    }
    return null
  } catch (error) {
    console.error(error)
    return null
  }
}
```

创建`src\common\interface.ts`

```ts
export interface LoginInfo {
  username: string
  code: string
}
```

新增了forget接口，传参option，类型`LoginInfo`，这是因为ts需要推导类型，如果以后参数有改变，这个`interface LoginInfo`也要修改

## 使用接口

`src\pages\forget.vue`

```ts
import { getCode, forget } from '@/api/login'
const submit = async (validate: any) => {
  const { valid } = await validate()
  if (!valid) {
    console.log('校验失败')
    return
  }
  forget({
    username: state.username,
    code: state.code,
  }).then((res) => {
    console.log(res)
    if (res.code === 200) {
      alert(res.msg)
    }
  })
}
```

- 把`username`和`code` post到后端，如果code是200，就弹窗发送成功，之前接口设计的时候`res.msg`是邮件发送成功
