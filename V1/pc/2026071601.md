# 封装alert&优化登录注册错误显示

本章主要封装alert组件，以及优化登录注册错误显示，

## 设计思路

登录这里，如果验证码错误，可以直接显示图片验证码错误，但是用户名或者密码错误，没必要说明是哪一个错误，所以这里就设置一个弹窗，告诉用户用户名或者密码错误就可以，点击确定或者页面任何地方都关闭弹窗

注册这里，当用户注册成功，可以弹窗注册成功并且跳转到登录

之前设置了一个`errorHandle`，服务器挂了，其实也可以弹窗一个服务器错误，请联系网站管理员

  cancelEvent?.()
  unmount()
}
//点击确定，执行回调事件，并且销毁组件
const success = () => {
  successEvent?.()
  unmount()
}
</script>

这个组件干了什么?

- 组件有一个alert和confirm，传入alert就显示一个确定按钮，传递其他参数就显示两个按钮，也就confirm

- 接收参数：msg、unmount、type和可选参数cancelEvent、可选successEvent

- 把接收的参数解构赋值：

  ```ts
  //3.5写法
  const { msg = '', unmount, type = 'alert', cancelEvent, successEvent } = defineProps<Props>()
  ```

  > 在 3.5 及以上版本中，当使用响应式 Props 解构时，可以自然地声明默认值。但在 3.4 及以下版本中，默认情况下并未启用响应式 Props 解构。为了用基于类型声明的方式声明 props 的默认值，需要使用 `withDefaults` 编译器宏；文档：https://cn.vuejs.org/api/sfc-script-setup.html#defineprops-defineemits

## 开发alert的组合式函数

src\components\modules\alert\index.tsx

```tsx
import { createApp } from 'vue'
import AlertComponent from './Alert.vue'
//导出myalert函数
export const myalert = (msg: string) => {
  //创建和挂载节点
  const root = document.createElement('div')
  document.body.append(root)
  //配置
  const options = {
    type: 'alert',
    msg: msg,
  }
  //动态创建vue组件
  const alertCom = createApp({
    setup() {
      //组件销毁
      const unmount = () => {
        alertCom.unmount()
        document.body.removeChild(root)
      }
      const newOptions = Object.assign(options, { unmount })
      return () => <AlertComponent {...newOptions} />
    },
  })
  alertCom.mount(root)
}
export const myconfirm = (msg: string, successEvent: () => void, cancelEvent: () => void) => {
  const root = document.createElement('div')
  document.body.append(root)
  const options = {
    type: 'confirm',
    msg,
    successEvent,
    cancelEvent,
  }
  const alertCom = createApp({
    setup() {
      const unmount = () => {
        alertCom.unmount()
        document.body.removeChild(root)
      }
      const newOptions = Object.assign(options, { unmount })
      return () => <AlertComponent {...newOptions} />
    },
  })
  alertCom.mount(root)
}
```

## 优化登录

```ts
//登录
const submit = async (value: any, actions: any) => {
  const { setErrors } = actions
  const result = await login({
    username: state.username,
    password: state.password,
    code: state.code,
    sid: useSidStore().sid,
  })
  //明确告知result就是HttpResponse类型
  const { code, msg } = result as HttpResponse
  if (code === 200) {
    state.username = ''
    state.password = ''
    state.code = ''
    router.push({ name: 'index' })
  } else if (code === 401) {
    setErrors({
      code: msg,
    })
  } else if (code === 404) {
    myalert(msg as string)
  }
}
```

登录成功后跳转到index路由并且清空输入框
