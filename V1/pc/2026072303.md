# 优化用户登录路由

本章主要实现用户登录后的路由跳转，之前这是一个简单的跳转到首页，但是头部没有改变状态，还是原来的登录注册，本章就是解决这个问题，但是也留下一个持久化问题，这在后续路由守卫那里解决

## 设计思路

- pinia里面添加userInfo
- 登录的时候，后端返回的data数据。通过setUserInfo设置userInfo
- header组件通过计算属性，pinia里面的userInfo填充数据
- 同样的，header组件通过计算属性，pinia里面的isLogin来判断显示头像昵称还是登录注册

## pinia添加userInfo

之前的pinia已经有分层优化了，具体看之前的章节

src\stores\user.ts

```ts
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state() {
    return {
      userInfo: {},
    }
  },

  getters: {},
  actions: {
    //设置用户基本信息
    setUserInfo(value: string) {
      this.userInfo = value
    },
  },
})

```

- 代码非常简单，就是state里面添加一个userInfo，actions添加一个设置userInfo的方法

## 登录组件

在登录的时候，后端返回了一个用户的data数据，需要把这个需要提交到userInfo上，这里就用到setUserInfo这个方法

src\pages\login.vue

```ts
const { code, msg, data } = result as HttpResponse
if (code === 200) {
    state.username = ''
    state.password = ''
    state.code = ''
    useUserStore().setUserInfo(data)
    router.push({ name: 'index' })
  }
```

- `useUserStore().setUserInfo(data)`关键语句

## 定义pinia的userInfo的类型

src\stores\user.ts

```ts
import { defineStore } from 'pinia'
interface UserInfo {
  name: string
  pic: string
  isVip: string
}
export const useUserStore = defineStore('user', {
  state() {
    return {
      userInfo: null as UserInfo | null,
    }
  },

  getters: {},
  actions: {
    //设置用户基本信息
    setUserInfo(value: UserInfo) {
      this.userInfo = value
    },
  },
})

```

这样定义之后，header组件就不会报错了，但是这还没有完成，再刚才登录的时候，这是去提交了userinfo，没有改变登录状态，所以接下来就需要在登录的时候改变登录状态

## 定义pinia的setisLogin的方法

src\stores\auth.ts

```ts
import { defineStore } from 'pinia'

export const useAuthStore = defineStore('auth', {
  //存数据的地方
  state() {
    return {
      sid: '',
      isLogin: false,
      token: '',
    }
  },
  //actions里面放置的是一个一个的方法，用于响应组件中的“动作”
  actions: {
    // 存放唯一标识
    setSid(value: string) {
      this.sid = value
    },
    //登录状态
    setisLogin(value: boolean) {
      this.isLogin = value
    },
  },

  getters: {},
})
```

登录组件使用

```ts
 if (code === 200) {
    state.username = ''
    state.password = ''
    state.code = ''
    useUserStore().setUserInfo(data)
    useAuthStore().setisLogin(true)
    router.push({ name: 'index' })
  } 
```

-  `useAuthStore().setisLogin(true)`关键代码
