# 首页Panel部分路由交互

## 优化ActiveClass

之前需要输入网址才能看到页面的不同，这当然不合理。所以现在优化一下Panel，把分类变成响应式数据。

```vue
<template>
  <div class="fly-panel fly-column">
    <div class="layui-container">
      <ul class="layui-clear">
        <li class="layui-hide-xs" :class="{ 'layui-this': $route.path === '/' }">
          <!-- <a href="/">首页</a> -->
          <RouterLink :to="{ path: '/' }">首页</RouterLink>
        </li>
        <li
          v-for="(item, index) in lists"
          :key="'panel' + index"
          :class="{ 'layui-this': $route.path === item.path }"
        >
          <RouterLink :to="item.path">
            {{ item.name }}
            <span class="layui-badge-dot" v-if="item.isNew"></span>
          </RouterLink>
        </li>

        <!-- 用户登入后显示 -->
        <li class="layui-hide-xs layui-hide-sm layui-show-md-inline-block">
          <a href="user/index.html">我发表的贴</a>
        </li>
        <li class="layui-hide-xs layui-hide-sm layui-show-md-inline-block">
          <a href="user/index.html#collection">我收藏的贴</a>
        </li>
      </ul>

      <div class="fly-column-right layui-hide-xs">
        <span class="fly-search"><i class="layui-icon"></i></span>
        <a href="jie/add.html" class="layui-btn">发表新帖</a>
      </div>
      <div
        class="layui-hide-sm layui-show-xs-block"
        style="margin-top: -10px; padding-bottom: 10px; text-align: center"
      >
        <a href="jie/add.html" class="layui-btn">发表新帖</a>
      </div>
    </div>
  </div>
</template>

<script lang="ts" setup>
import { reactive } from 'vue'

const lists = reactive([
  { name: '提问', path: '/index/ask', isNew: false },
  { name: '分享', path: '/index/askshare', isNew: false },
  { name: '讨论', path: '/index/discuss', isNew: true },
  { name: '建议', path: '/index/advuse', isNew: false },
  { name: '公告', path: '/index/notice', isNew: false },
  { name: '动态', path: '/index/logs', isNew: false },
])
</script>

<style></style>

```

首先来看看之前的样式;

```html
<li class="layui-hide-xs layui-this"><a href="/">首页</a></li>
<li><a href="jie/index.html">提问</a></li>
<li>
    <a href="jie/index.html"
       >分享<span class="layui-badge-dot"></span
        ></a>
</li>
<li><a href="jie/index.html">讨论</a></li>
<li><a href="jie/index.html">建议</a></li>
<li><a href="jie/index.html">公告</a></li>
<li><a href="jie/index.html">动态</a></li>
<li class="layui-hide-xs layui-hide-sm layui-show-md-inline-block">
    <span class="fly-mid"></span>
</li>
```

- layui-this是选中的样式，是**放在li标签**上的
- `RouterLink`最终渲染出来的是a标签，如果直接使用 `linkExactActiveClass: 'layui-this',`就会发现样式没有生效，因为样式在a标签上了，所以这里是`li`循环，`RouterLink`导航路由了，在`vue2`的时候可以在`RouterLink`添加`tag:li`,但是现在这个api已经被移除了，所以现在最终的解决方法是在li标签上加：`:class="{ 'layui-this': $route.path === '/' }"`，当路由导航到path路径时，激活`layuithis`这个样式
