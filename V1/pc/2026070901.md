# 前端项目初始化

本章主要对前端项目的初始化创建和基本架构搭建，主要是登录、注册、找回密码三个路由组件和头部底部组件

开始之前请下载fly社区的模板：https://wwbdz.lanzoum.com/i7oIV3vk6cve

```
npm create vue@latest
```

使用TypeScript、Router、Pinia、ESLint和Prettier，其他都否

安装依赖并启动开发服务器（不安装依赖直接用vsxode打开会代码会看到飘红，这是没依赖导致的，下载依赖就可以）

```shell
cd <your-project-name>
npm install
npm run dev
```

- vue3项目默认有vscode的设置，如果决定配置文件显示二级菜单可以取消掉：在.vscode\settings.json设置："explorer.fileNesting.enabled": false，
- 如果使用的之前插件，可以在.vscode\settings.json设置： "cSpell.enabled": false  这样就不会有很多蓝色的波浪线了

 :::

运行项目没有问题后剔除掉模板代码：

1. 把`src\components`文件夹下的所有组件删除

2. 把`src\assets`文件夹下的所有静态文件删除

3. 把`src\views`文件夹删除，并在`src`文件夹下创建`pages`文件夹，并创建`home.vue`,并创建`vue3base`骨架

4. 修改`src\router\index.ts`

   ```ts
   import { createRouter, createWebHistory } from 'vue-router'
   import HomeView from '@/pages/home.vue'
   const router = createRouter({
     history: createWebHistory(import.meta.env.BASE_URL),
     routes: [
       {
         path: '/',
         name: 'home',
         component: HomeView,
       },
     ],
   })
   export default router
   ```

5. 修改`src\App.vue`，内部全部代码清空，创建`vue3base`骨架

   ```vue
   <template>
     <RouterView></RouterView>
   </template>
   <script lang="ts" setup></script>
   <style></style>
   ```

6. 把`src\main.ts`的`import './assets/main.css'`删除

7. 运行项目：`npm run dev` ，此时查看控制台没问题，并且如果在`home.vue`写上文字，网页也能正常显示

8. 此时，项目src目录应该为以下状态：

   ```markdown
   ├─App.vue 
   ├─assets 
   ├─components 
   ├─main.ts 
   ├─pages 
   │ └─home.vue 
   ├─router 
   │ └─index.ts 
   └─stores 
     └─counter.ts 
   ```

## 创建组件

1. 在pages目录下创建`login.vue`、`reg.vue`和`forget.vue`三个组件，并创建`vue3base`骨架
2. 在components目录下创建header和footer两个公共组件，并创建`vue3base`骨架

## 组件样式

静态资源导入：把fly-maseter这个静态模板下的res文件夹下所有的文件复制到项目的assets目录；

在App.vue引入layui的样式代码

```ts
import '@/assets/layui/css/layui.css'
import '@/assets/css/global.css'
import '@/assets/layui/css/modules/layer/default/layer.css'
```

完成3个路由组件样式：

1. 把`html/user`中的`login.html`中的`layui-container`部分复制到`login.vue`的`template`中，模板中的{{code}}替换成验证码文字，把a连接替换成`routlink`
2. 把`html/user`中的`reg.html`中的`layui-container`部分复制到`reg.vue`的`template`中，模板中的{{code}}替换成验证码文字，把a连接替换成`routlink`
3. 把`html/user`中的`forget.html`中的`layui-container`部分复制到`forget.vue`的`template`中，模板中的{{code}}替换成验证码文字，把a连接替换成`routlink`

完成2个基础组件样式：

1. 把`html/common`中的`header.html`中全部复制到`header.vue`的`template`中，把a连接替换成`routlink`
2. 把`html/common`中的`footer.html`中全部复制到`footer.vue`的`template`中，把a连接替换成`routlink`

## 路由配置

把`login.vue`、`reg.vue`和`forget.vue`三个组件添加到路由

```ts
import { createRouter, createWebHistory } from 'vue-router'
import HomeView from '@/pages/home.vue'
import login from '@/pages/login.vue'
import reg from '@/pages/reg.vue'
import forget from '@/pages/forget.vue'
const router = createRouter({
  history: createWebHistory(import.meta.env.BASE_URL),
  routes: [
    {
      path: '/',
      name: 'home',
      component: HomeView,
    },
    {
      path: '/login',
      name: 'login',
      component: login,
    },
    {
      path: '/reg',
      name: 'reg',
      component: reg,
    },
    {
      path: '/forget',
      name: 'forget',
      component: forget,
    },
  ],
})

export default router

```
