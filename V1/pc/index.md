# PC端介绍

使用Vue3、Pinia、VueRouter、Axios、vee-validate等技术开发，本项目在window11开发，如遇到mac端的环境需要配置可以先问问AI

## 开发前准备

安装nvm：https://github.com/nvm-sh/nvm/releases

安装node：

```
nvm install v24.15.0
nvm use v24.15.0
```

验证：

```
node -v
npm -v
```

安装vscode：https://code.visualstudio.com/

配置vscode插件：https://wwbdz.lanzoum.com/iQ1pD3vg263c

安装git：https://git-scm.com/install/windows

配置git：

```
git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"
```

用于免密连接GitHub/Gitee：

```
ssh-keygen -t ed25519 -C "你的邮箱"
```
