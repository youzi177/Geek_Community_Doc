- 再头像上增加两个事件：`@mousemove="show" @mouseleave="hide"`
- `:class="{ 'layui-show': ishover }"` 通过ishover 控制是否显示
- 但是这样就会有一个bug，鼠标只要离开头像就立马隐藏，点不到菜单

## 进阶优化

给移入移出的时候加入延时器，然后把`@mousemove="show" @mouseleave="hide"`移到li标签上

```ts
//鼠标移入的时候显示
const show = () => {
  //清除定时器
  if (hoverCtrl.value) {
    clearTimeout(hoverCtrl.value)
  }
  hoverCtrl.value = setTimeout(() => {}, 200)
  ishover.value = true
}
//鼠标移出的时候隐藏
const hide = () => {
  //清除定时器
  if (hoverCtrl.value) {
    clearTimeout(hoverCtrl.value)
  }
  hoverCtrl.value = setTimeout(() => {
    ishover.value = false
  }, 500)
}
```

```vue
<!-- 登入后的状态 -->
<li class="layui-nav-item" @mousemove="show" @mouseleave="hide">
```

延时器让用户有足够的时间能移动到下拉菜单，而移动到菜单又是show状态
