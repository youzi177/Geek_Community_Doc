# 图片验证码校验和存储

本章主要实现验证码存储到redis中，并设置过期时间

## 验证码接口

```js
class publicController {
  async getCaptcha(ctx) {
    const svg = svgCaptcha.create();
    const body = ctx.request.query;
    // 保存验证码10分钟
    setValue(body.sid, svg.text, 10*60);
    ctx.body = {
      code: 200,
      data: svg.data,
    };
  }
}
```
