# 配置ESLint

本章主要给后端集成ESLint配置，有两种方式：

- eslint8+eslint-config-standard17（eslint-config-standard已经不再更新，但是周下载和使用人数基数依旧庞大）
- eslint9+neostandard（未来方向，本项目采用的）

## eslint8+eslint-config-standard17

安装：

```shell
npm install --save-dev eslint@8 eslint-config-standard eslint-plugin-import eslint-plugin-n eslint-plugin-promise
```

使用：.eslintrc.js

```js
module.exports = {
  env: {
    browser: true,
    es2021: true,
    node: true
  },

  extends: [
    'eslint:recommended',
    'standard'
  ],

  parserOptions: {
    ecmaVersion: 2022,
    sourceType: 'module'
  },

  rules: {
    'no-console': process.env.NODE_ENV === 'production'
      ? 'warn'
      : 'off',

    'no-debugger': process.env.NODE_ENV === 'production'
      ? 'warn'
      : 'off'
  }
}
```

保存格式化 .vscode\settings.json

```json
{
  "eslint.enable": true,

  "eslint.validate": [
    "javascript",
    "javascriptreact"
  ]
}
```

## eslint9+neostandard

安装：

```shell
npm install -D eslint neostandard
```

使用：eslint.config.js

```js
import neostandard from 'neostandard'

export default [
  ...neostandard({
    env: [
      'browser',
      'node'
    ]
  }),

  {
    rules: {
      'no-console': process.env.NODE_ENV === 'production'
        ? 'warn'
        : 'off',

      'no-debugger': process.env.NODE_ENV === 'production'
        ? 'warn'
        : 'off'
    }
  }
]
```
