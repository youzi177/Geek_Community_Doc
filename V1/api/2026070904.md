# 验证码接口

本章实现第一个简单的验证码接口，使用svg-captcha

## 安装使用svg-captcha

官方中文文档：https://github.com/produck/svg-captcha/blob/HEAD/README_CN.md

安装：

```
npm install --save svg-captcha
```

使用

`src\api\publicController.js`

```js
import svgCaptcha from 'svg-captcha';
class publicController {
  async getCaptcha(ctx) {
    const svg = svgCaptcha.create();
    ctx.body = {
      code: 200,
      msg: svg.data,
    };
  }
}
export default new publicController();

```

## 优化代码路由

根据上面的就已经完成了验证码接口，但是之前的demo示例需要改下，改为public前缀，同时添加路由前缀public

`src\router\routes.js`

```js
import combineRoutes from 'koa-combine-routers';
import publicRouter from './modules/publicRouter.js';

// 合并路由
export default combineRoutes(publicRouter);
```

`src\router\modules\publicRouter.js`

```js
import Router from 'koa-router';
import publictController from '../../api/publicController.js';
const router = new Router();
router.prefix('public');
// 获取图片验证码
router.get('/getCaptcha', publictController.getCaptcha);
export default router;
```
